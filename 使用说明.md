# OpenClaw Admin - 使用说明

## 🚀 快速开始

### 访问地址
```
http://10.168.1.155:5173
```

### 登录密码
```
wj12345
```

就这么简单！

---

## 📱 功能说明

登录后可以：
- 查看 Gateway 运行状态
- 监控会话统计
- 查看 Token 使用情况
- 监控系统资源（CPU、内存）
- 数据每 5 秒自动更新

---

## 🔐 安全说明

- 密码保存在浏览器本地
- 刷新页面不需要重新登录
- 点击右上角"退出登录"清除登录状态

---

## 💻 本地访问

如果在服务器本机访问：
```
http://localhost:5173
```

密码相同：`wj12345`
