# OpenClaw Admin - 登录说明

## 🔐 登录方式

### 设备 IP + 端口登录

**登录信息：**
- **设备 IP：** `10.168.1.155`
- **端口：** `7749`

### 登录步骤

1. 打开浏览器访问：`http://localhost:5173`
2. 输入设备 IP：`10.168.1.155`
3. 输入端口：`7749`
4. 点击"连接设备"按钮

### 登录后功能

- 查看 Gateway 运行状态
- 监控会话统计
- 查看 Token 使用情况
- 监控系统资源（CPU、内存）
- 实时数据更新

### 退出登录

点击右上角"断开连接"按钮即可退出。

---

## 🌐 远程访问

如果需要从其他设备访问管理系统：

1. 确保设备在同一局域网
2. 访问：`http://10.168.1.155:5173`
3. 输入设备 IP 和端口登录

---

## 🔧 修改设备信息

如果设备 IP 或端口变更，修改以下文件：

**前端登录验证：**
```typescript
// frontend/src/components/Login.tsx
if (values.ip === '10.168.1.155' && values.port === '7749') {
  // 修改这里的 IP 和端口
}
```

**后端端口：**
```bash
# backend/.env
PORT=7749
```
